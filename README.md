### # Overview

Introductory exercise to practise the basic Git workflow on a simple HTML project.

### # Assignment Summary

The exercise required to:

- Create a new project folder
- Open the project folder with VS Code
- Create a basic HTML file
- Create a new Git repository and perform the first commit
- Push the repository to a remote (e.g. GitHub)
- Continue developing the proposed layout incrementally, committing changes regularly
